# 💍 Jewellery Store Frontend

A responsive jewellery website built as a **mini-project** using **HTML, CSS, Bootstrap, and ReactJS**.

---

## Features

- Responsive design (works on mobile, tablet, desktop)
- Multi-page navigation (Home, Shop, About, Contact)
- Product listing with reusable `ProductCard` components
- Contact form with **JavaScript validation** (React state + alerts)
- Bootstrap-based modern layout (navbar, grid, buttons)
- Clean folder structure for scalability

- **HTML** – basic structure (`index.html` + JSX in React)

## Installation & Setup

1. Clone this repository:
```bash
   git clone https://github.com/your-username/jewellery-frontend.git
```

2. Navigate to the project folder:
```bash
   cd jewellery-frontend
```

3. Install dependencies:
```bash
   npm install
```

4. Start the development server:
```bash
   npm start
```

5. Open `http://localhost:3000` in your browser.
